# Tweendr

A tweet likes viewer and categorization app powered by Supabase.

## Features

- View and categorize your liked tweets
- Separate tabs for text and picture tweets
- Categorize tweets as GOOD or BAD
- Filter by category: UNSORTED, GOOD, or BAD
- Real-time counts and statistics
- Cloud-synced data via Supabase

## Setup

1. Set up your Supabase project and run `supabase_setup.sql`
2. Import your tweets using `importTweetsToSupabase.js`
3. Open `index.html` in your browser

## Tech Stack

- Vanilla JavaScript
- Supabase (PostgreSQL database)
- Modern CSS
